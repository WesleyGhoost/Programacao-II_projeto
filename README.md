Este projeto foi desenvolvido como parte da disciplina de Programação II, com o objetivo de aplicar conceitos fundamentais de desenvolvimento de software, incluindo lógica de programação, estruturação de código e possíveis integrações com banco de dados e/ou interface.

O sistema simula um cenário prático, permitindo a aplicação de boas práticas de desenvolvimento e organização de código.

🎯Objetivo:
- Aplicar conceitos aprendidos na disciplina
- Desenvolver um sistema funcional
- Praticar organização de código e versionamento com Git
- Simular um projeto real de desenvolvimento


🛠️ Tecnologias utilizadas:
- JavaScript
- Node.js
- Banco de Dados (Mongoose)


🚀Como executar o projeto
 🔧 Pré-requisitos
  - Ter o Node.js instalado
  - Ter um banco de dados Mongoosse configurado
